# MakerMind AI 网站原型

## 当前版本更新
- 网站内已放入 9 个项目的真实场景风格效果图
- 新增与扩展的图片资源已直接用于首页、生成器和作品效果页
- 生成器中的每个项目卡片会显示对应的“最终效果参考图”
- gallery.html 现在展示 9 个代表性项目
- 为后续 API 接入和视频接入预留了结构

## 主要页面
- index.html：首页
- rules.html：项目门槛
- generator.html：项目生成器
- gallery.html：作品效果展示页
- students.html：学生画像
